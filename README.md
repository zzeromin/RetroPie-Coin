# RetroPie-Coin
Simple Coin Acceptor for RetroPie using Photo Interrupter.

## Install
Install below:
<pre><code>cd /home/pi
git clone https://github.com/zzeromin/RetroPie-Coin.git
cd RetroPie-Coin/
chmod 755 setup.sh
sudo ./setup.sh
</code></pre>
